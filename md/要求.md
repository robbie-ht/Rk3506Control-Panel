# Lyra Shell - RK3506 控制面板

## 已实现功能

### 1. 主页系统信息显示
- 实时时钟显示（大字体）
- 日期显示（年月日星期）
- CPU 温度监控
- CPU 使用率监控
- 内存使用情况监控   （目前显示一直为0）待修复
- 系统运行时间显示
- 卡片式布局展示

### 2. 侧边栏导航
- 主页
- 设置
- 文件管理
- 模块化设计，方便添加新应用
- 选中状态高亮显示

### 3. 文件管理器（双栏模式，类似 MT 文件管理器）
- 左右双栏独立导航（左栏主目录，右栏根目录）
- 返回上级目录（工具栏按钮，自动跟踪当前激活面板）
- 长按/右键弹出操作菜单（复制、剪切、粘贴、删除、重命名、新建文件夹）
- 隐藏类型列，优先显示文件名，日期列紧凑显示
- 状态栏显示双栏文件统计

### 4. 插件系统
- AppInterface 接口支持
- 自动加载 plugins/ 目录下的 .so 插件
- 示例插件：Hello World

### 5. 主题系统
- 深色主题（深紫色系）
- 浅色主题（橙色+米白色系）
- 设置页面一键切换，重启后自动恢复
- 配置文件：`~/.lyrashell/theme.conf`
- 主题文件：`themes/dark.qss`、`themes/light.qss`（编译进二进制）
- 支持从外部 QSS 文件加载自定义主题

### 6. 界面设计
- 中文字体支持（文泉驿正黑）
- 右下角悬浮电源按钮（使用 power.png 图片） 所有图标都是用png,需要哪些列出来我来准备
- 全屏显示（适合嵌入式设备 1024×600）

### 7. 系统架构
- 模块化设计，全部解耦
- 插件系统支持（AppInterface 接口）
- 源代码按功能分类存放

## 编译运行

### x86 版本（本地测试）
```bash
cd /home/robbieht/桌面/Rk3506Control-Panel/build_x86
/home/robbieht/Qt5.12.9/5.12.9/gcc_64/bin/qmake ../LyraShell.pro
make -j$(nproc)
./LyraShell
```

### ARM 版本（RK3506）
```bash
cd /home/robbieht/桌面/Rk3506Control-Panel/build
/home/robbieht/Lyra-sdk/buildroot/output/rockchip_rk3506_luckfox/host/bin/qmake ../LyraShell.pro
make -j$(nproc)
```

```bash
cd helloworld
# x86
/home/robbieht/Qt5.12.9/5.12.9/gcc_64/bin/qmake helloworld.pro && make
# ARM
/home/robbieht/Lyra-sdk/buildroot/output/rockchip_rk3506_luckfox/host/bin/qmake helloworld.pro && make
```

## 工作记录与问题解决

### 2026-06-19 修复记录

#### 问题1: resources/power.png 资源路径不匹配
- **状态**: ✅ 已修复
- **描述**: resources.qrc 中路径为 `resources/power.png`，代码中加载路径为 `:/power.png`
- **解决**: 修改 `powerbutton.cpp` 中加载路径为 `":/resources/power.png"`

#### 问题2: 插件加载功能失效
- **状态**: ✅ 已修复
- **描述**: `AppManager::loadPlugins()` 从未被调用
- **解决**: 在 `lyramainwindow.cpp` 中添加 `loadPlugins` 调用

#### 问题3: 文件管理器改为双栏模式
- **状态**: ✅ 已完成
- **描述**: 用户要求类似 MT 文件管理器的双栏布局
- **解决**: 左右双栏独立导航，长按菜单操作，工具栏精简

#### 问题4: 回退上级目录按钮无效
- **状态**: ✅ 已修复
- **描述**: `m_btnUp` 初始化为 disabled，从未被启用
- **解决**: 在目录变化和面板切换时更新按钮状态

#### 问题5: 首页时间卡片太小
- **状态**: ✅ 已修复
- **解决**: 设置 `timeWidget->setMinimumHeight(160)`

#### 问题6: 文件管理器列显示优化
- **状态**: ✅ 已完成
- **解决**: 隐藏"类型"列，优先显示文件名

#### 问题7: 触屏长按不弹出菜单
- **状态**: ✅ 已修复
- **描述**: `customContextMenuRequested` 只响应鼠标右键，触屏无右键
- **解决**: 在 `FileListView` 中添加 `eventFilter`，600ms 长按触发菜单

#### 问题8: 主题切换功能
- **状态**: ✅ 已完成
- **描述**: 用户要求设置页面添加主题切换，重启后保持选择
- **解决**: 创建 `ThemeManager` 单例，深色/浅色主题 QSS 文件编译进资源，配置保存到 `~/.lyrashell/theme.conf`
